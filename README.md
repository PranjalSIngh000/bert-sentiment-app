# BERT Sentiment Analysis Web App

A simple web application that uses a BERT model to analyze sentiment of text input.

---

## 🚀 Features
- Sentiment prediction (Positive / Negative)
- Powered by DistilBERT
- Flask backend
- Simple frontend (HTML, CSS, JS)

---

## Project Structure

```text
bert-sentiment-app/
|
|--- backend/
|    |--- app.py # Flask application
|    |--- bert_model.py # BERT model logic
|
|--- frontend/
|    |--- index.html # UI structure
|    |--- script.js # Frontend logic
|    |--- styles.css # Styling
|
|--- requirements.txt # Dependencies
|--- README.md # Documentation
|--- .gitignore # Ignored files
```
